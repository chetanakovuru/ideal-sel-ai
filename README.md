# 🧠 Ideal Self AI

An AI prototype that helps you think like your future self.

## 💡 Idea
Most bad decisions don’t come from lack of knowledge — they come from emotions.

This project helps users:
- Understand their emotions
- See a clearer perspective
- Make better decisions
- Avoid sharing sensitive information

## ⚙️ Built With
- Python
- Streamlit

## 🚀 Features
- Emotion-aware responses
- Structured thinking guidance
- Privacy warning system
- Human-like decision support

## 🧪 Example Questions
- Why do I procrastinate?
- Is it safe to share personal information online?

## 📌 Note
This is a simple prototype built in a short time to explore the idea.
